# Steam Download Monitor

Python-скрипт для отслеживания скорости загрузки игр в Steam в реальном времени.

Скрипт работает в фоне, автоматически определяет путь установки Steam и показывает:
- какая игра сейчас скачивается
- текущий статус загрузки (DOWNLOADING / PAUSED)
- скорость загрузки

Информация выводится в консоль раз в минуту в течение 5 минут.

---

## Как работает

1. Определяет путь установки Steam через реестр Windows  
2. Находит все Steam-библиотеки (`libraryfolders.vdf`)  
3. Определяет активную загрузку через `steamapps/downloading`  
4. Вычисляет скорость загрузки по изменению размера файлов  
5. Определяет имя игры через `appmanifest_<appid>.acf`

---

## Требования

- Windows  
- Python 3.10+  
- Установленный Steam
